## Setting up Development Enviroment

Git Clone this repository in your working directory.
`git clone https://github.com/james-yang-dev/cinamon.git`

## Front

1. Move to cloned repository.

2. Install Libraries

```
yarn install
npm install
```

3. Run Server

```
yarn start
npm start
```
